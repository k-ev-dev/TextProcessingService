Не многопоточный метод описан в классе Processing.cs.
Многопоточный метод описан в классе MultiThreading.cs.

Исходники веб-сервиса в папке WcfService. 
Исходники консольного приложения в папке ServiceClient, 
там же находится файл для обработки (война и мир.txt) и файл-вывод (myoutput.txt).
